**발표자 멘트 가이드:**

**발표자 멘트 가이드:**

## 🟦 슬라이드 #5 — 국내 탄소 배출권 시장 현황

| 항목 | 내용 |
|------|------|
| **슬라이드 제목** | K-ETS 제4차 계획기간 진입 — 탄소의 가격이 오른다 |
| **슬라이드 목적** | 탄소 배출권 시장이 실제 비용 리스크의 원천임을 보여준다. |
| **핵심 메시지** | 2026년 제4차 배출권 거래제 시작, KAU25 톤당 12,000~13,000원이지만 5년 내 5만 원 돌파 전망. |

**주요 내용 Bullet:**
- 2026년 1월 1일부터 제4차 배출권 거래제 (2026~2030) 본격 진입
- '감축 가속화'와 '시장 원리 강화' 중심으로 제도 재편
- 현재 KAU25 톤당 약 12,000~13,000원, 전문가들은 5만 원 돌파 가능성 제기
- 유상할당 경매 본격화 → 기업 비용 부담 증가 불가피

**권장 시각 자료:**
- **KAU 가격 추이 차트** (최근 1~2년 + 5만 원 돌파 예측 구간 표시)
- 핵심 수치 강조 카드: "현재 ₩13,000 → 예측 ₩50,000"

**발표자 멘트 가이드:**
> "올해부터 제4차 배출권 거래제가 시작됩니다. 현재 톤당 만 이천 원 수준이지만, 유상할당 확대와 공급 축소로 5년 내 5만 원 돌파가 전망되고 있습니다. 기업 입장에서는 막대한 추가 비용이 발생하는 거죠."

## 🟦 슬라이드 #6 — 기존 데이터의 한계 → 우리 솔루션의 필요성

| 항목 | 내용 |
|------|------|
| **슬라이드 제목** | 기업 ESG 데이터의 현실 — 왜 AI 기반 대시보드가 필요한가 |
| **슬라이드 목적** | 현재 ESG 데이터 생태계의 한계를 보여주고, 우리 프로젝트의 존재 이유를 논리적으로 연결한다. |
| **핵심 메시지** | 정부 사이트는 기업별 조회형이라 대량 분석이 불가능하고, API 미제공으로 자동화가 안 되며, 기업마다 보고서 양식이 달라 비교가 어렵다. |

**주요 내용 Bullet:**
- 정부 공개시스템: 기업별 조회형 구조 → 대량 데이터 분석 불가
- API 미제공 → 자동화/파이프라인 구축 불가
- 기업마다 보고서 양식이 달라 → 비교 분석 난해
- 기업 간 비교·감축 추세 분석·배출권 전략 수립까지 지원하는 도구가 없음
- **→ 우리 솔루션: AI로 PDF를 구조화하고, 대시보드로 시각화·시뮬레이션**

**권장 시각 자료:**
- 환경정보공개시스템 화면 캡처 (조회형 한계 시각화)
- 기업별 보고서 양식 비교 캡처 (2~3개 기업)
- Before(현재) → After(우리 솔루션) 비교 다이어그램

**발표자 멘트 가이드:**
> "이런 상황에서 기업들은 데이터를 어디서 어떻게 관리할까요? 현실은 PDF 보고서를 일일이 열어보고 엑셀에 옮기는 수작업입니다. 정부 사이트도 검색형이라 비교 분석은 불가능합니다. 저희 프로젝트는 이 문제를 해결합니다."

## 🟧 슬라이드 #8 — AI 하이브리드 추출 파이프라인 (약 1분) [👑 추가됨]

| 항목 | 내용 |
|------|------|
| **슬라이드 제목** | 핵심 기술: AI 하이브리드 추출 엔진 |
| **슬라이드 목적** | 프로젝트의 핵심 차별성 중 하나인 '비용 효율적인 AI 파이프라인(Cost-Efficient Pipeline)'을 강조한다. |
| **핵심 메시지** | 모든 데이터를 비싼 Vision AI로 돌리지 않는다. Rerank 모델을 통해 꼭 필요한 곳에만 고정밀 검사를 수행하여 비용과 속도를 최적화했다. |

**주요 내용 Bullet:**
- 1차 구조화: Docling 엔진으로 전체 PDF 구조 분석 (Layout Analysis)
- 2차 텍스트 정제: BGE Reranker로 텍스트/표의 탄소 관련 연관성 평가 (Score 기반)
- 3차 하이브리드 추출:
  - 중요 표/이미지 ➡️ **GPT-4o Vision** (고정밀/고비용)
  - 일반 텍스트 ➡️ **GPT-Text** (중정밀/중비용)
  - 대량 정형 데이터 ➡️ **Regex(정규식)** (고속/저비용)
- **→ 결과: 추출 정확도 보장 + API 호출 비용 절감 극대화**

**권장 시각 자료:**
- Docling → Rerank → 3 갈래(Vision/Text/Regex)로 분기되는 **흐름도 다이어그램**
- 비용 절감 효과를 예시하는 아이콘 (비용 📉 / 속도 ⚡ / 정확 🎯)

**발표자 멘트 가이드:**
> "저희의 가장 큰 차별점은 비용 효율성입니다. 보고서 100페이지를 전부 비싼 GPT Vision 모델로 검사하면 파산하겠죠. 저희는 Reranker를 통해 연관성이 높은 페이지만 걸러내고, 꼭 필요한 핵심 레이아웃에만 Vision 모델을 투입합니다. 나머지는 빠르고 저렴한 텍스트나 정규식 처리로 설계했습니다."
